### Version

<img src="https://cloud.githubusercontent.com/assets/1725068/18940649/21945000-8645-11e6-86ed-4a0f800e5a73.png" width="32" height="32" /> <img src="https://cloud.githubusercontent.com/assets/1725068/18940648/2192ddba-8645-11e6-864d-6d5692d55717.png" width="32" height="32" />

if the passed picture is not already ``PNG``, it is internally converted (the alpha channel might be lost) 

output contains the same image in the following sizes: ``16, 32, 48, 64, 128, 256``

## Syntax

```
PICTURE TO ICO (picture;ico)
```

Parameter|Type|Description
------------|------------|----
picture|PICTURE|
ico|BLOB|

conversion is based simply on the [ICO](https://msdn.microsoft.com/en-us/library/ms997538.aspx) format specification.
